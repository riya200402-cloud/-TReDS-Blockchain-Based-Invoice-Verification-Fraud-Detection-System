# -TReDS-Blockchain-Based-Invoice-Verification-Fraud-Detection-System

Excited to present our Blockchain-powered FinTech project developed to enhance security, transparency, and trust within the TReDS (Trade Receivables Discounting System) ecosystem.

📌 Project Overview

This project was developed as part of our FinTech module with the objective of addressing critical challenges in receivables financing, including invoice duplication, fraudulent transactions, data manipulation, and limited transparency.

Using Blockchain Technology, the system securely stores invoice records in immutable blocks containing cryptographic hash values. Since every block is linked with the previous one, any unauthorized modification changes the hash instantly, enabling quick detection of fraudulent activities and ensuring complete data integrity.

Our solution aims to build a reliable and transparent ecosystem that strengthens trust among buyers, sellers, and financial institutions involved in TReDS operations.

🔑 Key Features

🔐 Blockchain-Based Invoice Verification
Each invoice is securely stored as a blockchain record containing:
• Invoice details
• Timestamp
• Current hash value
• Previous block hash

This creates a tamper-proof ledger that guarantees invoice authenticity and secure record management.

⚠️ Fraud Detection System
The platform uses cryptographic hash validation techniques to identify manipulated or duplicate invoices. Any alteration in invoice data automatically changes the hash value, enabling immediate fraud detection.

📊 Real-Time Invoice Status Tracking
Users can monitor invoice stages such as:
• Pending
• Verified
• Rejected

This improves workflow transparency and operational efficiency.

📧 Automated Email Notifications
Integrated Gmail API functionality to automatically send invoice verification and status update notifications to users.

🛡️ Immutable Financial Records
Blockchain technology ensures that stored invoice data cannot be altered without affecting the entire chain structure, providing secure and trustworthy financial record management.

💻 Interactive Dashboard
Designed a user-friendly interface for smooth invoice submission, verification, and tracking processes.
